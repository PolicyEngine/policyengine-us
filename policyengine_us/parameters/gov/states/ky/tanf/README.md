# Kentucky TANF (K-TAP) Parameters

This directory contains parameters for Kentucky's TANF program, known as K-TAP (Kentucky Transitional Assistance Program).

## Program Overview
K-TAP provides cash assistance to families with children who meet income and resource requirements.

## Parameter Files (To Be Implemented)
- `payment_standard.yaml` - Payment standards by assistance unit size
- `need_standard.yaml` - Need standards by assistance unit size
- `income/` - Income disregards and limits
