# GOV › STATES › MD › TAX › INCOME › DEDUCTIONS › ITEMIZED › PHASE_OUT Parameters

This section contains 3 parameters.

## Parameters

### `applies`
*Maryland itemized deduction phase-out applies*

Maryland phases out itemized deductions in the following years.

**Unit: bool | Period: year**

Current value (2025-01-01): **True**


### `rate`
*Maryland itemized deduction phase-out rate*

Maryland reduces itemized deductions by this rate of the amount over the phase-out threshold.

**Unit: /1 | Period: year**

Current value (2025-01-01): **0.075**


### `threshold`
*Maryland itemized deduction phase-out threshold*

Maryland phases out itemized deductions for filers with federal adjusted gross income above these thresholds.

**Unit: currency-USD | Period: year**
