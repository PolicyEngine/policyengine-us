# Kansas 2025 Individual Income Tax Implementation

This file tracks the implementation of Kansas 2025 individual income tax updates.

## Scope (Issue #7064)

1. Update disabled veteran exemption from $2,250 to $2,320 for 2025
2. Add 2025 references to all Kansas income tax parameters
3. Implement food sales tax credit sunset (returns 0 for 2025+)
4. Implement new Pregnancy Resource Act Credit (K-94) - 70% of charitable contributions
5. Add 2025 integration tests
